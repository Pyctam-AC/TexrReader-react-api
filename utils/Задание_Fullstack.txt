Предложить решение на NodeJs + Angular следующей задачи:
1. Есть некий файл на удаленном компьютере, логирующие работу оборудования. Пример файла во вложении. 
Имя файла формируется как "Move_" + штамп даты в формате ГГГГММДД + ".txt". Краткая структура файла:
- данные сохранены построчно, разделение по полям через символ ";"
- 1 поле штамп времени; 2 поле - тип события (START -запуск оборудования, STOP - остановка оборудования, SNAPSHOT - промежуточное сохранение данных между событиями START и STOP); 3 поле - количество обработанных единиц; 4 поле - средняя скорость обработки с последнего события START до текущего события; 5 поле - количество секунд с момента последнего события START; 6 - поле код причины останова.

2. Требуется по запросу пользователя или по таймауту читать файл и вычислять следующие данные:
- общее время работы (между первым событием START и последним STOP)
- общее время работы оборудования (между парами событий START и STOP)
- общее время простоя оборудования (между парами событий STOP и START)
- общее количество обработанных единици продукции
- средняя скорость обработки единиц в час
- максимальное время работы оборудования (между парами событий START и STOP)
- максимальное время простоя оборудования (между парами событий  STOP и START)
- формировать данные для построения графика производительности и обработки на основе поля 4 и 3 (нарастающим итогом) соответственно.

3. Визуализация данных
- на усмотрение разработчика можно использовать любые бесплатные фреймворки для визуализации по п.2.
